#  Scientific Calculator (Python + Tkinter)

This is a simple but powerful Scientific Calculator built using Python and Tkinter. It supports both basic arithmetic operations and advanced scientific functions in a clean graphical interface.

---

##  Features

- Basic operations: `+`, `-`, `×`, `÷`
- Scientific functions:
  - sin, cos, tan
  - log (base 10)
  - ln (natural log)
  - square root
  - power (^)
  - factorial
  - absolute value
- Parentheses support
- Backspace and clear buttons
- Keyboard input support
- Clean and modern GUI design

---

## How to Run

Just run the Python file and the calculator will start immediately.

```bash
python calculator.py
